TVmaze API

Welcome to the TVmaze API! Here you'll find answers to all things tv, including the name of that episode you really loved, the name of the actress that plays your favorite character, and more!

Let's do some searching!

Using this API, you'll be able to find out some specific information about your favorite tv shows and the people who make them possible! In order to access any information, we will first need to grab a specific ID.


Please enter the name of the show you'd like to learn more about in the bar below:


{{ tvShow.name }} - Show id: {{ tvShow.id }}

Click here to visit the show's page on the TVmaze website


Now that we have the show's id, we can use it to find some specific information about the show and the people who work on it!

Enter the id to see the cast list...


{{ cast.person.name }} - {{ cast.character.name }}

the crew list...

{{ crew.person.name }} - {{ crew.type }}

or a list of episodes!

Episode #{{ episodes.number }}: {{ episodes.name }} - Season {{ episodes.season }}

Looking to watch something else the actor who plays your favorite character has been in? Here, you'll be able to search for an actor and be linked direcly to the TVmaze profile, where you will be able to see some basic information, what else they've done, and more!

Please enter the name of the person you'd like to search for in the bar below:


{{ actor.person.name }}
